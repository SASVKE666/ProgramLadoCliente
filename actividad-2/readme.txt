# README - Formulario de Solicitud de Empleo

## Descripción
Formulario de solicitud de empleo

## Estructura del Proyecto
- `index.html`: Contiene la estructura HTML del formulario.
- `styles.css`: Contiene los estilos CSS para el formulario.
- `script.js`: Contiene la lógica de validación JavaScript.

## Instrucciones de Ejecución

### Requisitos Previos
https://www.justinmind.com/es/blog/ejemplos-de-formularios-web-movil/
https://elementor.com/blog/es/diseno-de-formularios-web-15-ejemplos-para-hacerlo-correctamente/
Asegúrate de tener un navegador web moderno instalado (como Google Chrome, Firefox o Edge).

### Pasos para Ejecutar el Proyecto

1. **Descargar el Proyecto:**
   - Clona o descarga este repositorio en tu máquina local.

2. **Abrir el Archivo HTML:**
   - Navega a la carpeta del proyecto y abre el archivo `index.html` en tu navegador web. Esto se puede hacer haciendo doble clic en el archivo o arrastrándolo a la ventana del navegador.

3. **Interacción con el Formulario:**
   - Completa los campos del formulario.
   - Asegúrate de seguir las reglas de validación (por ejemplo, el correo electrónico debe tener un formato válido).
   - Si hay errores, se mostrarán mensajes debajo de los campos correspondientes.

4. **Enviar el Formulario:**
   - Haz clic en el botón "Enviar Solicitud" para simular el envío del formulario. 
   - Si todo está correcto, los datos se enviarán (en un entorno real, se enviarán al servidor).
